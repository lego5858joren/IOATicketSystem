# Islands of Adventure Ticketing System

**I will be updating this with a GUI later on.

## Project Overview
This project is a Java-based ticketing system for Universal Orlando's Islands of Adventure. It allows users to purchase tickets for a specific date, choose between different types of Express Passes, and manage their visits to various rides within the park.

### Key Features
- **Ticket Purchase:** Users can purchase tickets for a specific date, ensuring tickets cannot be purchased for past dates.
- **Express Pass Options:** Users can choose between an unlimited Express Pass or a single-use Express Pass, which limits the use of the pass to one time per ride.
- **Ride Management:** Users can visit various rides within the park, with express pass holders skipping lines and non-pass holders waiting a random amount of time before entry.
- **Ride Tracking:** For single-use Express Pass holders, the system tracks which rides have been used to prevent multiple uses of the pass on the same ride.
